# Color Clicker
Color Clicker is a game which was inspired by the the very popular game OSU! from japan. The objective is to use your mouse to click the circles that appear on the screen as fast as possible to gain points.

How to play
Several circles spawn in random locations on the screen in a 3x3 grid. Using only your mouse, your job is to click as many circles as you can in 30 seconds. Each circle you click gives you 1 points.
